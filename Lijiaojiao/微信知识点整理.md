>1.better-scroll 影响click

使用了better-scroll的关东插件，其中的元素点击事件不生效，
更神奇的是，在PC谷歌测试是没有问题的，但是在微信客户端和微信工具测试的时候，点击时间总是不起作用
解决方法：初始化设置，在方法里面加上下面函数，或者method 配置里面click:true
```
 _initScroll() {
    this.boxScroll= new BScroll(this.$refs.wrapper, {
       click: true
    })
}
```
