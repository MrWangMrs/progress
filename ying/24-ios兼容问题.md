# 移动端:active伪类无效的解决方法
> 在iOS系统的移动设备中，需要在按钮元素或body/html上绑定一个touchstart事件才能激活:active状态。

```javascript
document.body.addEventListener('touchstart', function () { //...空函数即可});
```
