# IOS8 与 display:flex 的兼容性问题
>加-webkit-
>display: flex;
>display: -webkit-flex;
>justify-content: space-between;
>-webkit-justify-content: space-between;

# nginx 重定向 严格匹配 和 模糊匹配
>严格匹配(只能有hello)  
```javascript
    location  ~ /hello$ {
        rewrite ^.*$  https://www.baidu.com/;
    }
```
>模糊匹配(含有hello即可)
```javascript
    location  ~ /hello {
        rewrite ^.*$  https://www.baidu.com/;
    }
```
