

循环版

```js
let str = 'aaaa{{aaa}}aaaddg{{44}}fhjdr{{fdhgh}}'
let arr = str.split('{{')
let arr2 = []
arr.forEach(e => {
    let temp = e.match(/\S+(?=}})/g)
    temp && arr2.push(...temp)
})
console.log(arr2)
// [ 'aaa', 'gdfheth', 'fdhgh' ]
```

简易版(两种正则都取自vue源码)

```js
let str = 'aaaa{{aaa}}aaaddg{{44}}fhjdr{{fdhgh}}'
// 改进了，可以截取数字
let reg = /{{([^{][\s\S]+?|[\d]+?[^}])}}/g
// let reg = /\{\{((?:.|\n)+?)\}\}/g
// let reg = /{{(.+?)}}/g
let arr = str.match(reg)
arr = arr.map(e => e.replace(/{{|}}/g, ''))
console.log(arr)


```
